# SkillSphere 

SkillSphere is a simple online learning platform where users can explore different courses, see details, and improve their skills.

## 📌 Project Purpose

The goal of this project is to build a course platform using Next.js where users can:

* Browse courses
* View course details
* Register and login
* Access protected content after authentication

---

## 🌐 Live URL: https://skill-sphere-xi-gules.vercel.app/


---

## Key Features

* 🏠 Home page with hero section and popular courses
* 📚 All Courses page (dynamic data from JSON)
* 🔍 Course Details page (dynamic routing)
* 🔒 Protected Route (only logged-in users can access details)
* 🔐 Authentication system (Register & Login using Better Auth)
* 🎨 Level-based UI (Beginner / Intermediate / Advanced badges)
* 📱 Fully responsive design (mobile + desktop)
* 🧭 Active navbar highlighting

---

## Extra Sections

* 📌 Learning Tips section (study & productivity tips)
* 🏆 Top Instructors section (based on rating & level)

---

## 🛠️ Technologies Used

* Next.js (App Router)
* React.js
* Tailwind CSS
* DaisyUI
* Better Auth (Authentication)
* MongoDB (Database)
* React Icons



## 🔐 Environment Variables

Create a `.env.local` file in root:

BETTER_AUTH_SECRET=your_secret_key
BETTER_AUTH_URL=http://localhost:3000
MONGODB_URI=your_mongodb_connection_string

---

## ⚙️ Installation & Setup

1. Clone the repo:

```
git clone your-repo-link
```

2. Install dependencies:

```
npm install
```

3. Run the project:

```
npm run dev
```
